# Compilation à partir du Code Source

{% note warning %}

Cette page est en cours de traduction. Pour la documentation complète, veuillez consulter la version anglaise.

{% endnote %}

Ce guide couvre la compilation d'IceGate à partir du code source.

## Prérequis

- **Rust** >= 1.80.0
- **Cargo** (inclus avec Rust)
- **Git**

## Compilation

```bash
# Compilation debug
cargo build

# Compilation release
cargo build --release
```
