# Модель Данных

{% note warning %}

Эта страница находится в процессе перевода. Полную документацию смотрите в английской версии.

{% endnote %}

IceGate хранит данные наблюдаемости в четырёх таблицах Apache Iceberg.

## Обзор Таблиц

| Таблица | Описание | Сценарий Использования |
|---------|----------|------------------------|
| `logs` | LogRecords OpenTelemetry | Логи приложений |
| `spans` | Спаны распределённых трейсов | Трассировка запросов |
| `events` | Семантические события | Бизнес-события |
| `metrics` | Все типы метрик | Мониторинг производительности |

## Общие Паттерны

### Мультитенантность

Все таблицы используют партиционирование по `tenant_id`.

### Хранение Атрибутов

Атрибуты хранятся как `MAP(VARCHAR, VARCHAR)`.
