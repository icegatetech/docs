# Сборка из Исходного Кода

{% note warning %}

Эта страница находится в процессе перевода. Полную документацию смотрите в английской версии.

{% endnote %}

Это руководство охватывает сборку IceGate из исходного кода.

## Предварительные Требования

- **Rust** >= 1.92.0
- **Cargo** (входит в Rust)
- **Git**

## Сборка

```bash
# Debug сборка
cargo build

# Release сборка
cargo build --release
```
