# Конфигурация

{% note warning %}

Эта страница находится в процессе перевода. Полную документацию смотрите в английской версии.

{% endnote %}

IceGate использует YAML файлы конфигурации для настройки компонентов.

## Структура Файла Конфигурации

### Конфигурация Сервиса Query

```yaml
# query.yaml
loki:
  enabled: true
  host: "0.0.0.0"
  port: 3100

prometheus:
  enabled: true
  host: "0.0.0.0"
  port: 9090

tempo:
  enabled: true
  host: "0.0.0.0"
  port: 3200
```

## Переменные Окружения

| Переменная | Описание | По умолчанию |
|------------|----------|--------------|
| `AWS_ACCESS_KEY_ID` | Ключ доступа S3 | - |
| `AWS_SECRET_ACCESS_KEY` | Секретный ключ S3 | - |
| `AWS_REGION` | Регион S3 | `us-east-1` |
